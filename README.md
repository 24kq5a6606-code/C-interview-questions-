# C-interview-questions
What is C?
Answer: C is a general-purpose, procedural programming language developed by Dennis Ritchie in 1972. It is widely used for system programming, embedded systems, and application development.
2. What are the basic data types in C?
Answer: int, char, float, double, and void.
3. What is a variable?
Answer: A variable is a named memory location used to store data.
4.What is the difference between = and ==?
Answer:
= is the assignment operator.
== is the equality comparison operator.
5. What is a pointer?
Answer: A pointer is a variable that stores the address of another variable.
6.What is the difference between printf() and scanf()?
Answer:
printf() displays output.
scanf() reads input from the user.
7. What is an array?
Answer: An array is a collection of elements of the same data type stored in contiguous memory locations.
8. What is a function?
Answer: A function is a block of code that performs a specific task.
9. What is recursion?
Answer: Recursion is a process in which a function calls itself.
10.What is a structure?
Answer: A structure is a user-defined data type that groups variables of different data types.
